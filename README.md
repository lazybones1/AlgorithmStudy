## commit 방식
- 각자 이름으로 branch 생성
 - commit 후 pull request 요청
 ```
 git add 
 // 파일 추가
 git commit -m "add(문제난이도/문제번호/이름) : 소스파일.java"
 // 파일 변경
 git commit -m "fix(문제난이도/문제번호/이름) : 소스파일.java"
 git push origin '브랜치명'
 ```
